Connect Firebase

This board syncs in real time across all your devices using Firebase (free). Follow these steps once to get connected.

Step 1 — Create a Firebase project

Go to console.firebase.google.com → Add project → name it "Sunset Board" → disable Google Analytics → Create project.

Step 2 — Enable Realtime Database

In your project: Build → Realtime Database → Create database → choose US region → Start in test mode → Enable.

Step 3 — Get your config

Project Settings (gear icon) → Your apps → Add app → Web (</>) → Register → copy the firebaseConfig object values below.

Step 4 — Enter your config

Your config is saved in this browser. You'll need to enter it once per device (or edit the HTML to hard-code it). Firebase free tier supports up to 100 simultaneous connections — more than enough.
